🧮 Calculator Web Application

📌 Overview
 
The Calculator Web Application is a responsive web-based calculator developed using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic operations while demonstrating DOM manipulation, event handling, keyboard input handling, and error handling.

✨ Features

- Basic arithmetic operations (+, −, ×, ÷)
- Percentage (%) calculation
- Decimal number support
- Clear (AC) button
- Delete (DEL) button
- Keyboard input support
- Error handling for invalid expressions
- Responsive and user-friendly interface

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

🚀 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/Calculator-Web-App.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Calculator-Web-App
   ```

3. Open `index.html` in your preferred web browser.

🎯 Usage

- Enter numbers using the on-screen buttons or keyboard.
- Click operators (+, -, ×, ÷) to perform calculations.
- Press **=** or **Enter** to calculate the result.
- Use **AC** to clear the display.
- Use **DEL** or **Backspace** to delete the last character.
